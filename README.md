# DearDiary
## Android Diary App

### 구현
> MVVM Architecture    
> Realm    
> Observer Pattern with Live Data    

### 기능
> 일기 제목, 내용, 사진 추가와 수정 및 저장     
> 알람 설정    
> 위치 설정    
> 날씨 설정
